Alex-Taqueria Capstone Project

-Description

Alex-Taqueria is a Java console application that allows customers to create and customize taco orders. Customers can order Single Tacos, 3-Taco Plates, and Burritos. They can customize their order by selecting meats, cheeses, toppings, sauces, drinks, and chips & salsa.

The application calculates the total cost of the order, displays the order details, and saves a receipt to a receipts folder when the order is completed.

-Features

Create a new order

Order Single Tacos, 3-Taco Plates, and Burritos

Choose tortilla/shell type

Select meats, cheeses, toppings, and sauces

Add extra meat and extra cheese

Add drinks

Add chips & salsa

View complete order details before checkout

Automatic price calculation

Receipt generation with date and time stamp

Order validation to ensure customers order at least one item

-How to Run

Open the project in IntelliJ IDEA.

Run the Main.java file.

Select "New Order".

Follow the menu prompts to customize tacos, drinks, and chips & salsa.

Checkout to view the order total and generate a receipt.
